## 🕹️ Controles

- **Flechas ← →** o **A/D** - Mover patrulla
- **Toca la pantalla** - Controlar en móvil
- **Espacio** - Reiniciar

## ⭐ Features

- 🚗 Auto patrulla con luces reales
- 🛡️ Powerups: Escudo e invencibilidad
- ⭐ x2 puntos dobles
- 🎵 Música y efectos de audio generados
- 📱 Responsive en móviles
- 🏆 Sistema de récord (LocalStorage)
- 7️⃣ 7 niveles de dificultad progresiva

## 🎯 Objetivo

Esquiva las piedras el máximo tiempo posible. Cada nivel aumenta la dificultad. ¡Demuestra que eres el mejor!
